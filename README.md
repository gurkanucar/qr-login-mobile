### Used Packages

**sockjs-client & stompjs & net:** for connecting to stomp socket


### How to run

#### clone the project: https://github.com/gurkanucar/qr-login-mobile.git

```bash
  git clone https://github.com/gurkanucar/qr-login-mobile.git
  cd qr-login-mobile
```

 #### install the packages

```bash
  npm install
```

#### run
    
```bash
  yarn android
```
